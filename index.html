chapter30.~ば

A~ば B.It is used when
  • Only the condition A is met so that
  • Condition B can be fulfilled.
  • For Noun
    Noun+ ならば / じゃなければ

      Examples

    • 先生-> 先生 ならば / じゃなければ
      Noun.Teacher
    • 学生-> 学生 ならば / じゃなければ
      Noun.Student
    • 選手-> 選手 ならば / じゃなければ
      Noun.Player
  • For な Type Adjective
    な Type Adjective(without だ)+ ならば / じゃなければ

      Examples

    • 有名だ-> 有名 ならば / じゃなければ
      Adj.Famous
    • 暇だ-> 暇 ならば / じゃなければ
      Adj.idle
    • 上手だ-> 上手 ならば / じゃなければ
      Adj.well
    • 下手だ-> 下手 ならば / じゃなければ
      Adj.Clumsy
  • For い Type Adjective
    い Type Adjective(without い)+ ければ / くなければ

      Examples

    • 暑い-> 暑 ければ / くなければ
      Adj.warm
    • 寒い-> 寒 ければ / くなければ
      Adj.cold
    • 痛い-> 痛 ければ / くなければ
      Adj.sick
    • 遅い-> 遅 ければ / くなければ
      Adj.hurry
    • 眠い-> 眠 ければ / くなければ
      Adj.sleepy
  • For Verb
    • For type 1 Verb
      u type ending to e type ending+ ば / Verb ない form+ なければ

        Examples

      • 行く-> 行け ば / 行か なければ
        verb.to go
      • 描く-> 描け ば / 描か なければ
        verb.to draw
      • 入る-> 入れ ば / 入ら なければ
        verb.to enter
      • 買う-> 買え ば / 買わ なければ
        verb.to buy
    • For type ２ Verb
      without る ending+ れば / Verb ない form+ なければ

        Examples

      • 食べる-> 食べ れば / 食べ なければ
        verb.to eat
      • 寝る-> 寝 れば / 寝 なければ
        verb.to sleep
      • 出かける-> 出かけ れば / 出かけ なければ
        verb.to go out
    • For type 3 Verb

        Examples

      • 来る-> 来 れば / 来 なければ
        verb.to come
      • する-> す れば / し なければ
        verb.to do

    Another important Grammars
  • 気持ち vs　気分
    • 気持ち: 오감에 따른 심경
    • 気分: 정신적인 심경
    • 見る vs　見える
    • 見る: to see
    • 見える: to be seen
      聞く vs　聞こえる
    • 聞く: to listen to
    • 聞こえる: to be heard
  • Noun + の / Regular Verb form + ために: 위해서
    • ~まで: interval,distance
    • ~までに: determined date,time
  • Verb ます　Form +　の方 = A method to ~
  • Verb can work like adjective to nouns.
  • ~before : basic form+前に
  • ~After : た form+後で
  • planned object or action：basic form+Noun
    • 行くひと:갈 사람
    • 買う物: 살 물건
    • 行く所: 갈 장소
  • done object or action：た form+Noun
    • 行ったひと: 간 사람
    • 買った物: 산 물건
    • 行った所: 간 장소
  • 物　vs 事
    • 物 refers to things specific or not.
        Examples
      • 食べるもの:something to eat
    • 事 refers to action, or state.
        Examples
      • すること：something to do
  • また vs　まだ
    • また refers to more.
        Examples
      • またあした！:see you tomorrow!
    • まだ refers to yet.
        Examples
      • まだです：it is not yet.
